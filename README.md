# RankBloom – Premium SEO & Web Solutions

RankBloom is a high-performance, aesthetically driven web agency focused on helping small businesses scale through fast, SEO-optimized, and mobile-ready websites.

![Project Preview](https://github.com/devadex247/aiadebayo-co.git)

## 🚀 Technlogy Stack
- **Frontend:** Vanilla HTML5, CSS3 (Modern Flex/Grid), JavaScript (ES6+)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Animations:** Intersection Observer API, CSS Keyframes
- **Icons:** FontAwesome 6

## ✨ Key Features
- **Dynamic Mouse Spotlight:** Interactive cards that react to user movement.
- **Cinematic Video Backgrounds:** Slow-motion atmospheric headers for professional appeal.
- **Seamless Marquee:** Optimized testimonial scroll for social proof.
- **Chat Bloom AI:** Dedicated AI support interface for customer inquiries.
- **Responsive Core:** Mobile-first architecture ensuring perfect display on all devices.

## 🛠️ Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/devadex247/rankbloom.git
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```
